# ✈️ Travel & Tour Web Application (Next.js + TypeScript)

🔗 **Live Demo:** https://travel-tour-nextjs-ruby.vercel.app/

---

## 📌 Overview

A modern and responsive **Travel & Tour Web Application** built using **Next.js, TypeScript, and Tailwind CSS**.  
This project allows users to explore travel destinations with a clean, fast, and visually appealing UI.

The goal of this project is to create a scalable frontend travel platform with smooth user experience and modern design patterns.
This project also focuses on building a strong foundation for production-level frontend development with clean code architecture and maintainable structure.


---

## ✨ Features

- 🌍 Modern travel landing page UI
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🏝️ Destination showcase layout
- ⚡ Fast performance using Next.js
- 🧠 Type-safe development using TypeScript
- 🎯 Clean reusable components
- 🌐 Deployed on Vercel

---

## 🛠️ Technologies Used

- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **Vercel (Deployment)**
